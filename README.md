# Which code to choose?
- `All Batch.c` is the main file. Any batch's student can use it.
- `68-69.c` this is made for CSE-68 & 69 batches only. As it includes the **Tution & Registration Fees** of these particular batches.
- The files ending with `CR.c` are intended to be used by **batch CRs**. As CRs can write a post by copying the output and post it to groups.
# How to run?
- If you are on Windows, then compile the code in *Code::Blocks* or *Visual Studio Code* or whatever IDE you like.
- If you are on Mac, Linux or ...BSD then do the followings:
  ```
  $ gcc 'All Batch.c'
  $ .\a.out
  ```
  Here, inside the `''` is the C file's name.
# Core, GED & Lab Course
Find your course credits from Student Portal > Registered Course.
- Course Codes starting with CSE are Core Course. (PPS, DS, Electrical Cirsuits, etc.)
- Course Codes starting without CSE are GED Course. (English, Math, Physics, etc.)
- Course Names ending with Lab are Lab Course. (DS Lab, Physics Lab, etc.)
